# ☕ 欢迎来到 CoffeePop Studio！

你好！欢迎来到 CoffeePop Studio —— 一个由 Minecraft 玩家与开发者组成的创意工坊。

我们会在这里一起制作一些有趣的、关于 Minecraft 乃至其他领域的小东西。如果你对技术、创作或游戏本身感兴趣，欢迎常来逛逛！

## 🚀 我们在做什么

我们是一群热爱折腾的开发者，主要活跃在 **Java 生态** 和 **Paper 插件** 领域，但也乐于探索 Rust、前端、工具链等各种好玩的技术。

目前我们在孵化的项目包括（但不限于）：

| 项目 | 简介 |
| :--- | :--- |
| **[OmniCore](https://github.com/CoffeePopStudio/OmniCore)** | 全能日志与回滚插件，支持方块、容器、背包、末影箱全量追踪 + 现代网页控制台 |
| **[RustyJavaC-Native](https://github.com/CoffeePopStudio/RustyJavaC-Native)** | 通过 Rust 与 Java 原生交互，提供高性能底层能力 |
| **[RustyJavaC-Gradle-Plugin](https://github.com/CoffeePopStudio/RustyJavaC-Gradle-Plugin)** | 配合 Native 项目使用的 Gradle 插件，简化构建流程 |

更多有趣的项目正在酝酿中，欢迎保持关注！

## 🤝 加入我们

我们欢迎更多 **Minecraft 玩家 / 开发者 / 创作爱好者** 加入 CoffeePop Studio，一起做点有意思的事。

**加入方式：**  
在本组织的 **Discussion** 中发起申请，简单介绍一下你自己（擅长什么、想做什么），我们会综合评估后邀请你加入组织。

> 我们看重的不只是技术，还有热情与协作精神。哪怕你刚开始学编程，只要愿意学，我们也欢迎！
